## 🚀 2단계 - 지하철 구간 추가 리팩터링

### 인수 테스트 작성

- [] 구간 추가 제약사항 변경
    - 모든 구간에 구간에 신규 역 등록이 가능하다.
    - 상행선 그리고 하행선이 구간에 모두 등록되어 있지 않아야 등록이 가능하다.
    - 상행선 또는 하행선에 역이 한가지는 등록되어 있어야 등록이 가능하다.
    - 기존 구간 사이에 신규 역이 추가되면 기존 구간의 길이보다 작아야 한다.
- [] 노선 조회시 응답되는 역목록 정렬 순서 변경
    - 상행 종점 -> 하행 종점 순으로 반환한다.
